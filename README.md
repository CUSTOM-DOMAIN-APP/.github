# Custom Domain organization profile

This repository holds the organization-level GitHub metadata for Custom Domain, the managed platform that lets a SaaS product's users connect their own domains in one click at [customdomain.ai](https://customdomain.ai). GitHub renders the contents of [profile/README.md](profile/README.md) as the front page of this organization.

## What's in this repository

- [profile/README.md](profile/README.md): the organization front door, with a map of every repository in this org and where to start.
- Org-wide community health files, as they are added. GitHub treats files in a repository named `.github` as shared defaults for every repository in the organization.

If you are looking for the product, the docs, or the guides, start from the [organization profile](profile/README.md).

## About Custom Domain

Custom Domain is the managed domain-connection platform that lets a platform's users connect their own domain: support for 63 DNS and registrar providers through one-click provider authorization, API token connections, or a guided manual flow, with 25+ providers configured fully automatically, plus domain ownership verification and automatic TLS issuance and renewal. Every repository in this organization, including this one, is built and maintained by the Custom Domain team. Learn more at [customdomain.ai](https://customdomain.ai), read the [documentation](https://app.customdomain.ai/docs), or [sign up](https://app.customdomain.ai/signup).
